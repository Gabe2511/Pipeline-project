# Pipeline-project 📁 ⚙️

🛠 🎯

## Um projeto de pipeline ETL (Extract, Transform, Load) em cobrança de crédito utilizado em processos de engenharia de dados **extrai** dados de sistemas legados e birôs de crédito, **transforma** essas informações para limpar duplicidades e calcular riscos/scores de inadimplência, e **carrega** tudo em um Data Warehouse ou banco centralizado. Isso permite monitorar atrasos, prever inadimplência e automatizar campanhas de cobrança com eficiência.
